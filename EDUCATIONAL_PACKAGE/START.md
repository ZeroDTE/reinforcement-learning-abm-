# Start hier!

Willkommen zur Reinforcement Learning Semesteraufgabe.

## Los geht's

1. Stelle sicher dass Python 3.8+ installiert ist
2. 2. Installiere die Dependencies: `pip install -r requirements.txt`
   3. 3. Generiere die Notebooks:
     
      4. ```bash
         cd EDUCATIONAL_PACKAGE
         python create_notebooks.py
         ```

         Das erstellt das Student-Notebook `1_Student_Notebook.ipynb` mit Aufgaben.

         Wenn du auch das Loesungsbuch brauchst:

         ```bash
         python create_notebooks.py solution
         ```

         Das erstellt zusaetzlich `2_Solution_Notebook.ipynb`.

         4. Starte Jupyter: `jupyter notebook`
         5. 5. Oeffne `1_Student_Notebook.ipynb`
            6. 6. Lese die Anleitung und fuell die TODOs aus!
              
               7. ---
              
               8. ## Was du machen wirst
              
               9. Phase 1: Die GridWorld Umgebung verstehen
               10. Phase 2: Q-Learning Agent trainieren
               11. Phase 3: Deep Q-Learning mit Neuronalen Netzen
               12. Phase 4: Alles testen und Ergebnisse vergleichen
              
               13. Jede Phase dauert ca. eine 90-Minuten Sitzung.
